# 贪吃蛇（网页本地版）

## 运行

最简单：直接双击打开 `index.html`。

如果你发现键盘方向键/Space 不生效：先用鼠标点一下画布，再按键。

可选：用本地静态服务器（更接近生产环境）

```bash
cd projects/snake-web
python -m http.server 5173
```
然后打开：<http://127.0.0.1:5173>

## 操作

- 方向键 / WASD：移动
- Space：暂停/继续
- Enter：开始/重开
- S：暂停时单步（调试用）

## 难度

- 新手：更慢
- 简单：当前默认速度
- 困难：更快
- 极难：非常快

切换难度会影响“初始速度”，游戏过程中仍会随着吃到食物逐步加速。

## 特殊食物（10秒后消失，最后3秒闪烁）

- 黄色食物：吃到后 **10秒内** 红色食物变为 **2x2（4格）**，更容易吃到；且红色食物分值变为 **15分**。
- 灰色食物：吃到后 **扣 20 分**（最低扣到 0）。难度越高，灰色食物出现得越多。

## 功能

- 撞墙/撞自己结束
- 吃食物加分、变长、逐步加速
- 本地保存最高分（localStorage，不联网）
